1. Open your terminal to the Project Root folder. Make sure your terminal is pointing to the MCO2 folder (the one containing src, bin, and test.txt).

2. Compile the Code Run the following command to compile the Java files:
   javac -d bin src/bfs/\*.java

3. Run the Application Execute the program using the classpath flag pointing to the bin folder:
   java -cp bin bfs.Main

4. When inputting file location, include file name + file extension at the end.
   e.g. C:\Users\ACER\Downloads\CCDSALG-MCO2-main\CCDSALG-MCO2-main\FILE_NAME.txt
